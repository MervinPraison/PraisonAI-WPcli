# Server Command

Server profile management.

## Quick Start

```bash
# List servers
praisonaiwp server list
```

## Subcommands

### list

List configured servers.

```bash
praisonaiwp server list [OPTIONS]
```

### add

Add a server.

```bash
praisonaiwp server add NAME [OPTIONS]
```

### remove

Remove a server.

```bash
praisonaiwp server remove NAME [OPTIONS]
```

### set-default

Set default server.

```bash
praisonaiwp server set-default NAME [OPTIONS]
```

## Examples

```bash
# List servers
praisonaiwp server list

# Set default
praisonaiwp server set-default production
```
