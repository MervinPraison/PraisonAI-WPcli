# Shell Command

Interactive WordPress shell.

## Quick Start

```bash
# Start shell
praisonaiwp shell
```

## Usage

```bash
praisonaiwp shell [OPTIONS]
```

**Options:**

| Option | Description |
|--------|-------------|
| `--server TEXT` | Server name from config |

## Examples

```bash
# Start interactive shell
praisonaiwp shell

# Start shell on specific server
praisonaiwp shell --server production
```
