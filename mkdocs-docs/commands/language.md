# Language Command

Language pack management.

## Quick Start

```bash
# List installed languages
praisonaiwp language list
```

## Usage

```bash
praisonaiwp language [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `list` | List installed languages |
| `install` | Install language pack |
| `activate` | Activate language |

## Examples

```bash
# List languages
praisonaiwp language list

# Install language
praisonaiwp language install es_ES

# Activate language
praisonaiwp language activate es_ES
```
