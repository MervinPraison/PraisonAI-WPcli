# Taxonomy Command

Manage WordPress taxonomies.

## Quick Start

```bash
# List taxonomies
praisonaiwp taxonomy list
```

## Subcommands

### list

List registered taxonomies.

```bash
praisonaiwp taxonomy list [OPTIONS]
```

### get

Get taxonomy details.

```bash
praisonaiwp taxonomy get TAXONOMY [OPTIONS]
```

## Examples

```bash
# List taxonomies
praisonaiwp taxonomy list

# Get taxonomy details
praisonaiwp taxonomy get category
praisonaiwp taxonomy get post_tag
```
