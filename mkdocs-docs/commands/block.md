# Block Command

WordPress block management.

## Quick Start

```bash
# List registered blocks
praisonaiwp block list
```

## Usage

```bash
praisonaiwp block [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `list` | List registered blocks |

## Examples

```bash
# List all blocks
praisonaiwp block list

# List on specific server
praisonaiwp block list --server production
```
