# i18n Command

Internationalization (POT/PO/MO file management).

## Quick Start

```bash
# Generate POT file
praisonaiwp i18n make-pot
```

## Usage

```bash
praisonaiwp i18n [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `make-pot` | Generate POT file |
| `make-json` | Generate JSON translations |

## Examples

```bash
# Generate POT
praisonaiwp i18n make-pot

# Generate JSON
praisonaiwp i18n make-json
```
