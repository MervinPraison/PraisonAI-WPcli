# Post Type Command

Manage custom post types.

## Quick Start

```bash
# List post types
praisonaiwp post-type list
```

## Subcommands

### list

List registered post types.

```bash
praisonaiwp post-type list [OPTIONS]
```

### get

Get post type details.

```bash
praisonaiwp post-type get POST_TYPE [OPTIONS]
```

## Examples

```bash
# List all post types
praisonaiwp post-type list

# Get post type info
praisonaiwp post-type get post
praisonaiwp post-type get product
```
