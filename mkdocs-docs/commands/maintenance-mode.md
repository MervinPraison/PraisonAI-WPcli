# Maintenance Mode Command

Manage WordPress maintenance mode.

## Quick Start

```bash
# Check status
praisonaiwp maintenance-mode status

# Activate
praisonaiwp maintenance-mode activate
```

## Subcommands

### status

Check maintenance mode status.

```bash
praisonaiwp maintenance-mode status [OPTIONS]
```

### activate

Activate maintenance mode.

```bash
praisonaiwp maintenance-mode activate [OPTIONS]
```

### deactivate

Deactivate maintenance mode.

```bash
praisonaiwp maintenance-mode deactivate [OPTIONS]
```

## Examples

```bash
# Check status
praisonaiwp maintenance-mode status

# Activate
praisonaiwp maintenance-mode activate

# Deactivate
praisonaiwp maintenance-mode deactivate
```
