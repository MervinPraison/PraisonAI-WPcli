# Admin Command

WordPress admin management.

## Quick Start

```bash
# Get admin URL
praisonaiwp admin url
```

## Usage

```bash
praisonaiwp admin [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `url` | Get admin URL |
| `paths` | Show admin paths |

## Examples

```bash
# Get admin URL
praisonaiwp admin url

# Get admin paths
praisonaiwp admin paths
```
