# Profile Command

WordPress performance profiling.

## Quick Start

```bash
# Profile WordPress
praisonaiwp profile stage
```

## Usage

```bash
praisonaiwp profile [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `stage` | Profile by stage |
| `hook` | Profile by hook |

## Examples

```bash
# Profile stages
praisonaiwp profile stage

# Profile hooks
praisonaiwp profile hook
```
