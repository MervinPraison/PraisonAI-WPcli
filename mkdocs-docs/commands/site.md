# Site Command

WordPress site management.

## Quick Start

```bash
# Get site info
praisonaiwp site info
```

## Subcommands

### info

Get site information.

```bash
praisonaiwp site info [OPTIONS]
```

### option

Manage site options.

```bash
praisonaiwp site option get KEY
praisonaiwp site option set KEY VALUE
```

## Examples

```bash
# Get site info
praisonaiwp site info

# Get option
praisonaiwp site option get blogname

# Set option
praisonaiwp site option set blogname "My Site"
```
