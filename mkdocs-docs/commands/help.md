# Help Command

Get help for commands.

## Quick Start

```bash
# General help
praisonaiwp help

# Command help
praisonaiwp help create
```

## Usage

```bash
praisonaiwp help [COMMAND]
```

## Examples

```bash
# General help
praisonaiwp help

# Specific command help
praisonaiwp help create
praisonaiwp help update
praisonaiwp help ai
```
