# Embed Command

WordPress embed management.

## Quick Start

```bash
# List embed providers
praisonaiwp embed list
```

## Usage

```bash
praisonaiwp embed [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `list` | List embed providers |

## Examples

```bash
# List providers
praisonaiwp embed list
```
