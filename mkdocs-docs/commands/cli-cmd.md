# CLI Command

WP-CLI management and utilities.

## Quick Start

```bash
# Check WP-CLI version
praisonaiwp cli version
```

## Usage

```bash
praisonaiwp cli [SUBCOMMAND] [OPTIONS]
```

**Subcommands:**

| Subcommand | Description |
|------------|-------------|
| `version` | Get WP-CLI version |
| `info` | Get WP-CLI info |

## Examples

```bash
# Get version
praisonaiwp cli version

# Get info
praisonaiwp cli info
```
