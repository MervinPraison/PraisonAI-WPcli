# Super Admin Command

WordPress multisite super admin management.

## Quick Start

```bash
# List super admins
praisonaiwp super-admin list
```

## Subcommands

### list

List super admins.

```bash
praisonaiwp super-admin list [OPTIONS]
```

### add

Add super admin.

```bash
praisonaiwp super-admin add USER [OPTIONS]
```

### remove

Remove super admin.

```bash
praisonaiwp super-admin remove USER [OPTIONS]
```

## Examples

```bash
# List super admins
praisonaiwp super-admin list

# Add super admin
praisonaiwp super-admin add admin

# Remove super admin
praisonaiwp super-admin remove user
```
